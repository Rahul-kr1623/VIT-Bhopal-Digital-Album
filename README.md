## âœ¨ Features

* **Interactive Drag & Drop:** Users can freely drag and toss photo cards around the screen using mouse or touch gestures.
* **Lightbox Gallery:** Click on any photo to view it in full-screen mode with custom captions.
* **Background Ambience:** Integrated audio player for background music with a toggle (Play/Pause) button.
* **Video Memory:** A dedicated video card with a custom player (no autoplay) and download option.
* **Shuffle Mechanism:** A "Shuffle Memories" button that randomly scatters the photos again if the desk gets too messy.
* **Download Functionality:** Direct download links for both high-resolution images and videos.
* **Responsive Design:** Optimized for both desktop and mobile viewing.
